# Security Policy

## Reporting a vulnerability

Please do not report security vulnerabilities in a public issue.

Use the repository's **Security** tab and select **Report a vulnerability** to
send the report privately. Include the affected animation or file, steps to
reproduce the issue, its potential impact, and any suggested mitigation.

You can expect an initial response within seven days. Please allow time for a
fix to be prepared before disclosing the vulnerability publicly.
